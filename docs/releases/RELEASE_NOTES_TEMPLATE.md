# SeptaZip v1.0.0

Native macOS archive manager powered by the official 7-Zip engine.

## Highlights

- Browse archive contents in a native macOS app
- Compress and extract common archive formats
- Finder right-click integration for extract, test, and compress actions
- Modernized app icon and refreshed macOS UI
- Bundled official 7-Zip `7zz` engine

## Download

- Download `SeptaZip-1.0.dmg`
- Verify it with `SeptaZip-1.0.dmg.sha256`

## Install

1. Open the DMG
2. Drag `SeptaZip.app` into `Applications`
3. Launch the app once
4. Enable `SeptaZip Finder Extension` in:
   `System Settings -> Privacy & Security -> Extensions -> Finder Extensions`

## Notes

- This is a direct-download macOS build, not a Mac App Store release.
- If the build is not notarized, macOS may show a Gatekeeper warning on first launch. If that happens, open:
  `System Settings -> Privacy & Security`
  then choose `Open Anyway`.
