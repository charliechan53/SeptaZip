# 7-Zip Source Tree

This folder stores the upstream 7-Zip source used to build `7zz` for the macOS app.

Current source location:
- `source_code/7zip/Asm`
- `source_code/7zip/C`
- `source_code/7zip/CPP`
- `source_code/7zip/DOC`

Use `source_code/sync_7zip_source.sh` to refresh this source from an upstream git ref.
